# Test media

Place `test.mp4` here for the Phase 1 RTSP test server.

Requirements:
- Contains English/Latin licence plates visible for at least 1 second each
- Ground truth written to `ml/benchmarks/phase1_gt.json`
- Recommended: 720p or 1080p, 15–30 fps, 30–120 seconds

See `ml/benchmarks/download_test_data.sh` to fetch a suitable CC-licensed sample.
